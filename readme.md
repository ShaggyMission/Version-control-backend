# API Topics

API REST desarrollada con Node.js y Express para la gestión y consulta de temas (`topics`).

## Funcionalidades

- Obtener todos los temas registrados.
- Consultar un tema específico mediante su ID.
- Manejo de respuestas 404 cuando un recurso no existe.
- Pruebas automatizadas utilizando Jest y Supertest.
- Contenerización mediante Docker.

## Tecnologías utilizadas

- Node.js
- Express.js
- Jest
- Supertest
- Docker
